# Mission Control AI

Durante uma missão espacial, diferentes informações precisam ser analisadas continuamente, como temperatura, comunicação, bateria, oxigênio e estabilidade operacional. A partir desses dados, um sistema computacional pode identificar riscos, emitir alertas e sugerir ações para apoiar a tomada de decisão.
